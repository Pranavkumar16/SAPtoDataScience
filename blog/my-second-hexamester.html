My Second Hexamester in Data Science

Date: 2025-04-25

Overview

This was a fast-paced and incredibly enriching term. I was introduced to key concepts in probability, statistics, machine learning, and econometrics—all with a hands-on approach using R and R Markdown. I really enjoyed the journey and learned a lot of new things!

Topics Covered

  • Introduction to Probability and Random Variables
  • Inferential Statistics
  • Collecting and Displaying Data
  • Econometrics & Data Analysis – including an exciting report on house price comparisons using econometric methods.
  • Big Data and Machine Learning
  • Naïve Bayes Classification
  • Supervised & Unsupervised Machine Learning

My Favorite Part

My favorite part of this hexamester was writing a full report in R Markdown to compare house prices using econometric techniques. It was a rewarding challenge that deepened my understanding of real-world data analysis and statistical reporting.

Tools and Skills Acquired

  • R for statistical analysis
  • LaTeX for professional report writing and math notation
  • Mathematical formulation of models and hypotheses
  • R Markdown for dynamic reporting
